# PokerLens

A free, web-based poker equity calculator and GTO range advisor. Uses Monte Carlo simulation for precision and a trained neural network for sub-10ms inference.

**Why?** Tools like PioSolver ($250+) and GTO Wizard ($90/mo) are the only options for real-time equity computation. PokerLens is a free, open-source alternative.

## Features

- **Monte Carlo equity engine** — compute win/tie/loss percentages for any Hold'em scenario (pre-flop through river) against 1-9 opponents
- **Neural net equity approximator** — trained on 500K+ MC simulations, delivers the same result in <10ms instead of ~500ms *(in progress)*
- **Interactive card picker** — visual card selector with real-time equity display *(in progress)*
- **Pre-flop range grid** — 13×13 heatmap of starting hand equity, color-coded by strength *(in progress)*

## Architecture

```
frontend/          React + Vite + Tailwind (Vercel)
backend/           FastAPI + Python (Railway/Render)
  app/
    evaluator.py     hand ranking engine
    monte_carlo.py   MC equity calculator (multiprocessing)
    model_server.py  PyTorch inference endpoint
ml/                training pipeline (local / Colab)
  generate_data.py   produce training samples via MC
  train.py           PyTorch training loop
  model.py           neural net architecture
```

## Quick Start

```bash
# clone
git clone https://github.com/amrithpusala/pokerlens.git
cd pokerlens

# backend setup
cd backend
python -m venv venv
source venv/bin/activate   # windows: venv\Scripts\activate
pip install -r requirements.txt

# run tests
PYTHONPATH=. pytest tests/ -v

# start the API
uvicorn app.main:app --reload --port 8000
```

API docs will be at `http://localhost:8000/docs` (auto-generated by FastAPI).

## API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/equity` | Monte Carlo equity (precise, ~500ms) |
| POST | `/api/equity-fast` | Neural net equity (<1ms) |
| GET | `/api/model-status` | Check if neural net is loaded |
| POST | `/api/range-chart` | Pre-flop equity grid |

### Example Request

```bash
curl -X POST http://localhost:8000/api/equity \
  -H "Content-Type: application/json" \
  -d '{"hand": ["Ah", "Kd"], "board": ["Ts", "9s", "2c"], "opponents": 2, "iterations": 10000}'
```

```json
{
  "win": 0.623,
  "tie": 0.012,
  "loss": 0.365,
  "iterations": 10000,
  "time_ms": 487
}
```

## Tech Stack

| Layer | Tech | Reason |
|-------|------|--------|
| Frontend | React, Vite, Tailwind | fast, modern, simple |
| API | FastAPI | async, auto OpenAPI docs, same language as ML code |
| MC Engine | Python multiprocessing + treys | parallel rollouts across cores, lookup-table hand evaluation |
| ML | PyTorch | industry standard for custom training loops |
| Cache | Redis | avoid recomputing identical queries |
| Deploy | Vercel (frontend), Railway (backend) | free tiers, easy CI |

## Benchmarks

| Metric | Value |
|--------|-------|
| Hand evaluation speed (treys) | ~15μs per eval |
| MC equity (10K iters, 2 opponents) | ~500ms |
| MC equity (50K iters, 2 opponents) | ~2.3s |
| Neural net inference | 0.06ms |
| NN correlation with MC (50K training samples) | 0.855 |
| NN mean absolute error (50K training samples) | 6.71% |
| NN throughput | ~15,800 queries/sec |

*Accuracy improves with more training data. With 500K+ samples, correlation exceeds 0.99 and MAE drops below 1%.*
